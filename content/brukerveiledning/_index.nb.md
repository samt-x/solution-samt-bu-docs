## Rask orientering

SAMT-BU Docs er bygget på Hugo og publiseres automatisk til Cloudflare Pages via GitHub Actions. Innhold redigeres på tre måter:

1. **Innebygd redigering** (anbefalt for fagpersoner) – visuell nettleserbasert editor tilgjengelig via «Endre»-menyen
2. **GitHub** – direkte redigering i Markdown-editoren på github.com
3. **Lokalt** – klon repo, rediger Markdown-filer, push

Innhold fra flere repoer slås sammen til ett nettsted. Endringer publiseres automatisk ca. 1 minutt etter at de er lagret.
